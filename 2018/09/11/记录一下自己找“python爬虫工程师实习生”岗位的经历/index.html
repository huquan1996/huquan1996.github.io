记录一下自己找“python爬虫工程师实习生”岗位的经历

发表于 2018-09-11 | 本文总阅读数 次
字数统计: 787 | 阅读时长 ≈ 3

今天是2018年9月11号，是我实习的第二天，记录一下找工作的历程，本来准备昨晚写的，但是第一天来实习，有点累了，就早点睡了，早上醒了，洗漱一下就来写这篇文章了。

鄙人的水平很低，而且还是一名大四的学生，所以投的都是python爬虫工程师实习生的岗位，投简历都是在网上投的，自己用的有拉勾，实习僧，大街，前程无忧，智联招聘……找实习的话，还是比较推荐实习僧这个网站，其他的也可以尝试的投一投，陆陆续续的投了54家公司，有8个公司打电话过来表示安排面试，还有几家在招聘网站上表示有意向不过还没有打电话过来，简历回馈有1/7吧，不过我只面试了2次

第一个面试是一个上海的公司

  • 电话邀约：9.3 10点
  • 面试开始时间：下午2：10
  • 经过时间：12分钟
  • 面试问题：
1. 先问了实习的时间，三个月，每周4天
2. 问java和python的区别
3. 怎么解析字段
4. 用过哪些代理
5. 用过哪些验证码识别
6. 基本的算法，问给你10000个数，里面有重复的，问怎么去重
7. ......

这家公司hr第一次打电话过来的时候，先是介绍了自己是某某公司的hr，其他的什么也没说，就让我去上海，这让我感觉很不好，怎么也要问一些基本的情况，看是否符合吧，像这样的行为，我觉得是对应聘者的不负责任

第二家面试的是一家合肥的公司

这家公司面试了2次，一次是技术面，一次是hr面试

技术面一共是53分钟（我问为什么记得这么准确，是因为进行的是视频面试，随着技术的进步，现在不少公司不反对以这样的方式进行面试）面试的问题就不细写了，大部分是按照职位要求和简历来问的，还有问了一些我不懂的，面试官很好，不知道的都和我解释了原理

二面是hr面试，问的不是技术类，先让自我介绍，之后就问了一些基本的情况，之后问还有什么问题，我问了上班时间，薪资待遇和培养
计划

合肥这家公司是我愿意去的，因为离家很近，而且还提供住宿，所以面试之前都是准备的很充分，面试的时候状态也很不错，之后运气很不错的通过了面试，收到了offer

之后还有几家公司来电话，但是决定去合肥这家公司了，所以之后的都婉拒了

总的来说，从自学python，爬虫到现在，大概有四个月的时间，借此找到一个实习的工作，而且自己也比较满意，觉得自己的运气还是不错的。

扫一扫关注我哦
